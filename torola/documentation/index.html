OnScroll Navigation (header, footer)

By default Header and footer elements follow scroll, they disappear on scroll down and appear on scroll up. To disable this functionality remove .sticky-header and .sticky-footer classes from bodyelement. You can remove both or just .sticky-footer class.

If you want header and footer elements always fixed use following classes

Dark/Light Navigation (header, footer)

By default navigation is dark. To enable light navigation add .light-navigation class to body

Horizontal Sliders

For Slide item widths use classes .vw-20, .vw-25 ... .vw-95, .vw-100. For example .vw-50 means 50% of window width.

To enable keyboard and mousewheel control use data-keyboard and data-mousewheel attributes.

Video Backgrounds

Use following attributes to setup video background. It supports Youtube, Vimeo and Local videos.

Attr Description
data-src Video URL (Youtube, Vimeo or Local video)
data-poster Poster Image URL. It will appear on mobile devices
data-controls Display Controls (true/false)
data-autoplay Autoplay video (true/false)
data-loop Enable loop (true/false)
data-volume Volume 0-10 (integer)

Thumbnail Videos

Thumbnail Video element will load original video (iframe) only after clicking play button. It supports Youtube, Vimeo and Local videos

Portfolio Grid

Use following attributes to setup Portfolio grid.

Attr Description
data-columns Column count
data-gutter Space between items
data-cell-ratio Default item aspect ratio

To Change item sizes use .wide and .tall classes

Masonry

Use following attributes to setup Masonry Portfolio.

Attr Description

data-gutter Space between items

OnScroll Animations

Add appear animations with data-animation attribute. Set animation-name, duration and delay like in the example below.

You can use 4 predefined animation names fade-in-bottom fade-in-left fade-in-right fade-in-top.

Animate.css library is also supported.

Spacing Classes

Class Names like .px-5 .mb-0 .my-10 are Bootstrap Spacing Classes. Sizes are extended up to 15.

Backgrounds & Overlays

Set Backgrounds with data-background attribute. Set image URL, HEX or RGB color codes


Use data-overlay attribute to set overlay color and opacity (separated with semicolon)
